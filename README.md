# abc-store

Monorepo интернет-магазина на:

- React + Vite + TypeScript + Tailwind
- Node.js + Express + Prisma
- PostgreSQL
- pnpm workspace

## Структура

- `apps/web` — клиент
- `apps/api` — сервер
- `packages/shared` — общие типы, схемы, константы
- `packages/config` — общие конфиги

## Запуск

Пока в процессе настройки.
